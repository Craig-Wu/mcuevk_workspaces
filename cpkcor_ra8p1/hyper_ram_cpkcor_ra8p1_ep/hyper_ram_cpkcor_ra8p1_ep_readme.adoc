== 驱动文件

在 src 文件夹下，有 `hyper_ram.c` 和 `hyper_ram.h`，其中包含了 HyperRAM 的操作。HyperRAM 在 RA8P1 上可以通过内存映射的方式访问，因此只有一个初始化函数，以及定义内存映射的起始地址和 HyperRAM 的大小，方便程序内使用

[source,c,subs="verbatim,attributes"]
----
#define HYPER_RAM_MAP_START_ADDR	0x70000000
#define HYPER_RAM_MAP_END_ADDR		0x78000000
#define HYPER_RAM_SIZE				(1024 * 1024 * 32)

#ifdef __cplusplus
extern "C" {
#endif

void HyperRAM_Init(void);

#ifdef __cplusplus
}
#endif
----

== HyperRAM 测试

对 HyperRAM 的测试位于 `src/test/test_hyper_ram.c` 中，入口函数为 `TestHyperRAM()`，该函数会检查 HyperRAM 并测试读写速度。只需在 hal_entry() 中完成对 HyperRAM 的初始化后调用即可

[source,c,subs="verbatim,attributes"]
----
HyperRAM_Init();
TestHyperRAM(HYPER_RAM_MAP_START_ADDR, HYPER_RAM_SIZE);
----
