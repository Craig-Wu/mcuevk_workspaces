== 驱动文件

相关驱动在 `src/lcd` 文件夹中，除此之外还有两张图片资源用于测试。

[.text-center]
--
image::Snipaste_2026-06-23_16-51-24.png[role="thumb"]
--

ST7796U 支持多种接口，因此需要对驱动型号和接口进行指定，在当前工程中，都已指定好。可以在以下位置查看：

* 在 `src/lcd/lcd.h` 中指定驱动芯片

[source,c,subs="verbatim,attributes"]
----
#define LCD_DRIVER_UNSELECT		0
#define LCD_DRIVER_EK79001D		1
#define LCD_DRIVER_ILI9881C		2
#define LCD_DRIVER_ST7796U		3
#define LCD_DRIVER_TFP410		4

#ifndef LCD_DIRVER
#define LCD_DIRVER				LCD_DRIVER_ST7796U
#endif
----

* 在 `src/lcd/lcd_st7796u.c` 中指定 ST7796U 的接口

[source,c,subs="verbatim,attributes"]
----
#define ST7796U_INTERFACE_8080_18BIT        0x00
#define ST7796U_INTERFACE_8080_09BIT        0x01
#define ST7796U_INTERFACE_8080_16BIT        0x02
#define ST7796U_INTERFACE_8080_08BIT        0x03
#define ST7796U_INTERFACE_3SPI              0x05
#define ST7796U_INTERFACE_MIPI              0x06
#define ST7796U_INTERFACE_4SPI              0x07
#define ST7796U_INTERFACE_RGB               0x10

#ifndef __ST7796U_INTERFACE
#define __ST7796U_INTERFACE     ST7796U_INTERFACE_4SPI
#endif
----

== 初始化屏幕和简单测试

* 声明一个 `LCD_Device` 设备

[source,c,subs="verbatim,attributes"]
----
static LCD_Device s_lcd;
----

* 设置方向，然后初始化。

[source,c,subs="verbatim,attributes"]
----
s_lcd.orientation = SCREEN_ORIENTATION_VERTICAL;
LCD_PortInit(&s_lcd);
----

在 `src/test` 文件夹中有对屏幕的简单测试，用于测试屏幕本身及相关 API 是否正常，该工程在初始化屏幕后，随即进行了测试，以显示彩条

[source,c,subs="verbatim,attributes"]
----
TestLCD(&s_lcd);
----
