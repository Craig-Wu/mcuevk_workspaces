== 驱动文件

OV5640 相关的驱动文件为 `src/ov5640.c` 和 `src/ov5640.h`，除此之外，还依赖 `src/iic.c` 和 `src/iic.h` 中提供的 IIC 读写函数。OV5640 提供两种接口，CSI 和 DVP，在 FSP 中分别对应 VIN 和 CEU 两种外设，此工程已设置好 OV5640 使用的接口

* 在 `src/ov5640.h` 中指定接口

[source,c,subs="verbatim,attributes"]
----
#define OV5640_INTERFACE_CSI	1
#define OV5640_INTERFACE_DVP	2

#ifndef OV5640_USING_INTERFACE
#define OV5640_USING_INTERFACE	OV5640_INTERFACE_CSI
#endif
----

* 随后，调用 OV5640 的初始化并指定输出分辨率，然后启动捕获。注意：设置的分辨率应与 VIN 中的设置一致

[source,c,subs="verbatim,attributes"]
----
OV5640_Init(1024, 600);
OV5640_StartCapture(NULL);
----

* 之后，可以使用 `OV5640_IsImageReady(uint8_t **p_image)` 检查是否获取到了完整的一帧图像。如果已有一帧完整的图像，形参会返回图像数据的地址

[source,c,subs="verbatim,attributes"]
----
while (OV5640_IsImageReady(&image) == 0) {
    R_BSP_SoftwareDelay(10, BSP_DELAY_UNITS_MILLISECONDS);
}
----
